Задача B. Поиски тропы

Имя входного файла: input.txt
Имя выходного файла: output.txt
Ограничение по времени: 1 секунда
Ограничение по памяти: 256 Mb

Зона огорожена и тщательно охраняется, вход в Зону разрешён только по пропускам и только сотрудникам Международного института внеземных культур. Однако сталкеры проникают в Зону. Опытный сталкер знает все безопасные тропинки в Зоне. Тропинки петляют по Зоне, образуя связный граф, в котором N - количество вершин и K - количество ребер. В целях безопасности группа сталкеров хочет построить максимальное покрытие этого графа непересекающимися по дугам путями. Ваша задача - помочь им найти минимально возможное количество путей в таком покрытии.

Входные данные

В первой строке входного файла задаются два целых числа N - количество вершин и K - количество ребер ( 2 ≤ N ≤ 1000000 ). Далее следует K строк, в каждой из которой задается дуга номерами связанных этой дугой вершин. Вершины нумеруются с единицы.

Выходные данные

В результирующий файл Вы должны вывести единственное число - ответ на задачу.
